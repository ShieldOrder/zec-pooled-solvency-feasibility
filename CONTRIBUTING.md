# Contributing

This repository is an adversarial feasibility hub.

Contributions are welcome when they improve rigor or falsifiability.

### In scope
- Cryptographic soundness analysis
- Threat model critique
- Deterministic verification review
- Negative test vector design
- Reorg and finality analysis

### Out of scope
- Product framing
- Yield or incentive discussion
- Wallet integration
- Governance advocacy
- Scope expansion beyond stated constraints

All contributions should assume that failure is an acceptable and informative outcome.
